Occupational Specialism

For the Occupational Specialism, the task is to analyse the problem, design a solution, create a prototype, and gathering and acting on feedback from users. This page will help break down each phase of the OS so it can be easier to understand.


Task 1 a(i) - Research the topic


Task 1 a(ii) - Write the proposal for your solution.


Task 1 b - Create a design for your solution.


Task 2 - Create your prototype and test it.


Task 3 a - Create a review form and gather information from it.


Task 3 b - Use the feedback given to you to evaluate how to better the system.
