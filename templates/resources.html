Programming content to focus on

There are lots of languages to learn when in this field and lots of content to learn in those languages so knowing what to cover is useful.

For year 1 of this course, the focus should be to developing basic skills in python like creating functions and using libraries and string manipulation. This will greatly help for tasks 2 and 4a. The two libraries you should work on in python are Pandas and MatPlotLib. Pandas is used for controlling and manipulating data and MatPlotLib is used for displaying this data. In pandas, you should be able to understand how to single out different types of data and group them together and order them well. For MatPlotLib, you should be able to make a range of tables with this data and label the graph.

For year 2 of this course, the focus should be on developing moderate skills in HTML, CSS and two other languages. It is recommended you use python and SQL. For python, you can use Flask to control how your templates (html files) are launched and how they can be effected using variables. You should also use SQLite3 which helps control databases for a website. In SQL, you should be able to understand how to create tables that contain data and link together using primary and foreign keys.



Fundamental skills

For both years, there are a lot of skills that are useful and benefit them and your future careers. Getting these down greatly helps you for your future and in these exams

One really good skill to learn, is how to evaluate your work and recieve feedback and work from it. These are extremely useful skills for the whole course, such as: ESP task 4b, OS task 3b and the core papers. Using this skill efficientlty can rack up lots of in areas that can potentially move you up a grade.

Another fundamental skill, is the ability to manage your time. This is good for ESP task 1 and the whole of the OS as they are rather big tasks with lots of time and it can be very easy to find this daunting or be relaxed. It is best to use the time given to you to the best of your ability without worrying too much about the time given.
